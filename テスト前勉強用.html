Webクリエイター能力試験 スタンダード_竹内先生期末テスト対策

4択テスト用 勉強資料

この資料について

この資料では、HTML・CSSの基本、画像、フォーム、レスポンシブ対応など、 Webページ制作に必要な基礎知識を整理しています。

テストでは、この資料に書かれている内容を中心に出題します。

1. HTMLの基本構造

HTMLは、Webページの内容や構造を作るための言語です。 HTML文書では、最初にDOCTYPE宣言を書き、その下にHTML全体の構造を記述します。

基本の形

<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>ページタイトル</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>見出し</h1>
  <p>本文です。</p>
</body>
</html>

覚えること

内容 コード 意味
DOCTYPE宣言 <!DOCTYPE html> HTML5の文書であることを示す
文字コード <meta charset="UTF-8"> 日本語を正しく表示する
ページタイトル <title> ブラウザのタブなどに表示される
本文 <body> ブラウザに表示される内容を書く

2. CSSの読み込み

CSSは、Webページの見た目を整えるための言語です。 外部CSSファイルを読み込むには、<link> 要素を使います。 

<link rel="stylesheet" href="style.css">
記述 意味
rel="stylesheet" CSSファイルであることを示す
href="style.css" 読み込むCSSファイルの場所を指定する

3. 画像の表示

画像を表示するときは、<img> 要素を使います。

<img src="photo.jpg" alt="写真">
属性 意味
src 画像ファイルの場所を指定する
alt 画像が表示されないときの代替テキスト
覚える： 画像は <img src="画像ファイル" alt="説明">

4. リンク

別のページやサイトに移動するリンクを作るには、 <a> 要素を使います。 

<a href="about.html">会社概要</a>
要素・属性 意味
<a> リンクを作る要素
href リンク先を指定する属性

5. 見出し・段落・改行

見出し

最も大きな見出しは <h1> です。

<h1>ページのタイトル</h1>

段落

文章のまとまりは <p> で表します。

<p>これは段落です。</p>

改行

改行だけを入れたいときは <br> を使います。

東京都<br>
新宿区
要素 意味
<h1> 最も重要な見出し
<p> 段落
<br> 改行

6. リスト

順序のないリスト

順番に意味がないリストには、<ul> と <li> を使います。

<ul>
  <li>りんご</li>
  <li>みかん</li>
  <li>バナナ</li>
</ul>

順序付きリスト

番号付きのリストには、<ol> と <li> を使います。

<ol>
  <li>HTMLを書く</li>
  <li>CSSを書く</li>
  <li>ブラウザで確認する</li>
</ol>
要素 意味
<ul> 順序のないリスト
<ol> 順序付きリスト
<li> リストの項目

7. HTMLのコメント

HTMLの中にメモを書きたいときは、コメントを使います。 コメントはブラウザには表示されません。 

<!-- ここはコメントです -->
覚える： HTMLのコメントは <!-- コメント -->

8. HTML5の構造を表す要素

HTML5では、ページの役割に合わせた意味のある要素が使えます。

要素 役割
<header> ヘッダー部分
<nav> ナビゲーション
<main> 主要コンテンツ
<section> 内容のまとまり
<article> 独立した記事
<footer> フッター部分

9. id属性とclass属性

HTMLでは、要素に名前を付けるために id や class を使います。 CSSで特定の要素にスタイルを当てるときにも使います。

id属性

id は、ページ内で原則として一意にします。 CSSで指定するときは # を使います。 

<div id="main">メイン部分</div>

#main {
  color: red;
}

class属性

class は、複数の要素に同じ名前を付けることができます。 CSSで指定するときは . を使います。 

<p class="text">文章1</p>
<p class="text">文章2</p>

.text {
  color: blue;
}
属性 CSSでの書き方 特徴
id #id名 ページ内で一意
class .class名 複数の要素に使える

10. CSSの基本

CSSは、HTMLの見た目を指定するために使います。

CSSの基本形

セレクタ {
  プロパティ: 値;
}

例

p {
  color: red;
}

この場合、すべての <p> 要素の文字色が赤になります。

11. よく使うCSSプロパティ

目的 CSSプロパティ 例
文字色を変える color color: red;
背景色を変える background-color background-color: #f5f5f5;
文字サイズを変える font-size font-size: 16px;
文字を中央揃えにする text-align text-align: center;
外側の余白 margin margin: 20px;
内側の余白 padding padding: 20px;
枠線 border border: 1px solid #000;
幅 width width: 300px;
角を丸くする border-radius border-radius: 10px;
非表示にする display: none; display: none;

12. marginとpaddingの違い

margin と padding はどちらも余白ですが、場所が違います。

プロパティ 意味
margin 要素の外側の余白
padding 要素の内側の余白 
.box {
  margin: 20px;
  padding: 10px;
}
覚える： 外側が margin、内側が padding

13. Flexbox(期末テスト対象外)

Flexboxは、要素を横並びにしたり、中央揃えにしたりするときに使うCSSのレイアウト方法です。

Flexboxを使う基本

.container {
  display: flex;
}

横方向の中央揃え

.container {
  display: flex;
  justify-content: center;
}
指定 意味
display: flex; Flexboxを有効にする
justify-content: center; 主軸方向に中央揃えする

14. box-sizing(期末テスト対象外)

box-sizing: border-box; を指定すると、 padding や border を含めて幅や高さを計算します。 

.box {
  width: 300px;
  padding: 20px;
  border: 1px solid #000;
  box-sizing: border-box;
}
覚える： box-sizing: border-box; は、paddingやborderを含めて幅を計算する。

15. 疑似クラス :hover(期末テスト対象外)

マウスカーソルが要素の上に乗ったときのスタイルを指定するには、 :hover を使います。 

a:hover {
  color: red;
}
疑似クラス 意味
:hover マウスが乗ったとき

16. CSSの優先順位(期末テスト対象外)

CSSでは、同じ要素に複数のスタイルが指定された場合、どの指定が優先されるかが決まっています。

同じ強さなら後に書いたCSSが優先

p {
  color: blue;
}

p {
  color: red;
}

この場合、後に書かれた color: red; が優先されるため、文字色は赤になります。

セレクタの強さ

idセレクタ > classセレクタ > 要素セレクタ > 全称セレクタ

セレクタ 例 優先度
idセレクタ #main 高い
classセレクタ .text 中
要素セレクタ p 低い
全称セレクタ * さらに低い

17. フォーム(期末テスト対象外)

フォームは、ユーザーが情報を入力するための部品です。

入力欄

<input type="text">

メールアドレス入力欄

<input type="email">

複数行の入力欄

<textarea></textarea>

プルダウンメニュー

<select>
  <option>東京</option>
  <option>大阪</option>
  <option>福岡</option>
</select>

ラベルの関連付け

<label for="name">名前</label>
<input type="text" id="name">
部品 HTML
入力欄 <input>
メール入力欄 <input type="email">
複数行入力欄 <textarea></textarea>
プルダウン <select> と <option>
ラベルの関連付け label for と input id

18. 画像形式

Webページでは、画像形式の特徴を理解して使い分けることが大切です。

画像形式 特徴
JPEG 写真に向いている
PNG 透明部分を扱える
覚える： 写真はJPEG、透明部分がある画像はPNG。

19. レスポンシブ対応とviewport(期末テスト対象外)

スマートフォンやタブレットなど、画面サイズが違う端末に対応することを レスポンシブ対応といいます。

スマートフォン表示に対応するためには、<head> 内にviewport設定を書きます。 

<meta name="viewport" content="width=device-width, initial-scale=1.0">
内容 意味
width=device-width 端末の画面幅に合わせる
initial-scale=1.0 初期表示倍率を1.0にする
