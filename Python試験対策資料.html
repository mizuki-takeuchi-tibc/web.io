この資料の使い方

この資料はPythonの基礎試験対策として、重要事項を 要点・コード例・注意点・練習問題 の形で整理したものです。

試験でよく問われる観点
  • 実行結果がどうなるか

1. 数値計算と基本演算

1.1 算術演算子

演算子 意味 例 結果
+ 加算 3 + 2 5
- 減算 3 - 2 1
* 乗算 3 * 2 6
/ 除算 5 / 2 2.5
// フロア除算 5 // 2 2
% 剰余 5 % 2 1
** べき乗 2 ** 3 8

1.2 / は常に浮動小数点数を返す

print(4 / 2)
print(type(4 / 2))

出力：

2.0
<class 'float'>
整数同士の割り算でも、/ の結果は float になります。

1.3 // と %

print(17 // 3)
print(17 % 3)

出力：

5
2

次の関係が成り立ちます。

17 == 3 * (17 // 3) + (17 % 3)

1.4 負の数のフロア除算に注意

print(-7 // 3)
print(-7 % 3)

出力：

-3
2
// は0に近い方向への切り捨てではありません。 小さい方向、つまり床関数的に処理されます。

1.5 剰余の利用例

偶数・奇数判定

n = 10

if n % 2 == 0:
    print("偶数")
else:
    print("奇数")

素数判定

n = 17

for i in range(2, n):
    if n % i == 0:
        print("素数ではない")
        break
else:
    print("素数")

1.6 代入演算子

i = 0
i += 1

print(i)

これは次と同じ意味です。

i = i + 1
書き方 意味
x += 1 x = x + 1
x -= 1 x = x - 1
x *= 2 x = x * 2
x /= 2 x = x / 2
x //= 2 x = x // 2
x %= 2 x = x % 2

2.1 文字列の基本

シングルクォートでもダブルクォートでも文字列になります。

2.2 文字列はイミュータブル

文字列は作成後に中身を書き換えられません。

word = "Python"
word[0] = "J"

これはエラーです。

文字列を変更したい場合は、新しい文字列を作ります。

print(new_word)

print(r"C:\new\test")

3.1 リストの基本

nums = [1, 2, 3, 4]
words = ["apple", "banana", "orange"]

3.2 リストはミュータブル

nums = [1, 2, 3]
nums[0] = 100

print(nums)

出力：

[100, 2, 3]

3.3 リストの連結とネスト

a = [1, 2]
b = [3, 4]

print(a + b)

出力：

[1, 2, 3, 4]
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]

3.4 主要メソッド

append(x)

末尾に1要素を追加。

a = [1, 2]
a.append([3, 4])
print(a)
[1, 2, [3, 4]]

extend(L)

別のリストなどの要素をまとめて追加。

a = [1, 2]
a.extend([3, 4])
print(a)
[1, 2, 3, 4]

insert(i, x)

指定位置に挿入。大きすぎる位置なら末尾。

a = [1, 2, 4]
a.insert(2, 3)
print(a)
[1, 2, 3, 4]

remove(x)

指定した値の最初の要素を削除。

a = [1, 2, 3, 2]
a.remove(2)
print(a)
[1, 3, 2]

pop(i)

指定位置の要素を削除し、その値を返す。

a = [10, 20, 30]
x = a.pop(1)
print(x)
print(a)
20
[10, 30]

sort()

元のリストをその場で並べ替える。

a = [3, 1, 2]
result = a.sort()
print(a)
print(result)
[1, 2, 3]
None

3.5 sort() の応用

words = ["Python", "AI", "program"]
words.sort(key=len)

print(words)

出力：

['AI', 'Python', 'program']
nums = [3, 1, 2]
nums.sort(reverse=True)

print(nums)

出力：

[3, 2, 1]
remove() は値を指定します。 pop() は位置を指定します。 この違いは試験でよく問われます。

4.8 match 文

    case 200:
        print("OK")
    case 404:
        print("Not Found")
    case _:
        print("Other")

出力：

case _ はワイルドカード

case _:
    print("その他")

| によるパターン統合

status = 403

match status:
    case 401 | 403 | 404:
        print("Not allowed")
    case _:
        print("Other")

ガード条件 if

x = 10

match x:
    case n if n > 0:
        print("正の数")
    case n if n < 0:
        print("負の数")
    case _:
        print("ゼロ")

5. コレクション：辞書・集合・タプル

5.1 辞書 dict

person = {
    "name": "Tanaka",
    "age": 20
}

print(person["name"])
print(person["age"])

出力：

Tanaka
20

追加・変更

person = {"name": "Tanaka"}

person["age"] = 20
person["name"] = "Suzuki"

print(person)

get()

person = {"name": "Tanaka"}

print(person.get("age"))
print(person.get("age", 0))

出力：

None
0

辞書のループ

scores = {"Alice": 80, "Bob": 90}

for name, score in scores.items():
    print(name, score)

ループ中に辞書サイズを変更しない

辞書をループしている最中に削除や追加をしてサイズを変えると、エラーになることがあります。

安全な書き方：

users = {
    "Alice": "active",
    "Bob": "inactive"
}

for name, status in users.copy().items():
    if status == "inactive":
        del users[name]

リストから重複を削除

nums = [1, 2, 2, 3, 3, 3]

空集合は set() で作ります。 {} は空の辞書です。

6.1 関数の定義

def greet():

6.2 引数と戻り値

def add(a, b):
    return a + b

出力：

6.3 return がない場合

def hello():
    print("Hello")

x = hello()
print(x)

出力：

Hello
None
関数が明示的に値を返さない場合、戻り値は None です。

6.4 デフォルト引数

def greet(name="Guest"):
    print("Hello", name)

greet()
greet("Alice")

出力：

Hello Guest
Hello Alice

デフォルト引数は定義時に評価される

i = 5

def f(x=i):
    print(x)

i = 10
f()

出力：

5

可変オブジェクトをデフォルト引数にしない

def add_item(x, items=[]):
    items.append(x)
    return items

print(add_item(1))
print(add_item(2))
print(add_item(3))

出力：

[1]
[1, 2]
[1, 2, 3]
デフォルト引数のリストが共有されるため、前回の結果が残ります。

安全な書き方：

def add_item(x, items=None):
    if items is None:
        items = []
    items.append(x)
    return items

6.5 *args

任意個の位置引数を受け取ります。関数内ではタプルになります。

def total(*args):
    print(args)
    return sum(args)

print(total(1, 2, 3))

出力：

(1, 2, 3)
6

6.6 **kwargs

任意個のキーワード引数を受け取ります。関数内では辞書になります。

def show_info(**kwargs):
    print(kwargs)

show_info(name="Alice", age=20)

出力：

{'name': 'Alice', 'age': 20}

6.7 / と * による引数制限

/ より前は位置専用引数

def func(a, b, /):
    print(a, b)

func(1, 2)

次はエラーです。

func(a=1, b=2)

* より後はキーワード専用引数

def func(a, *, b):
    print(a, b)

func(1, b=2)

次はエラーです。

func(1, 2)

7. ファイル操作とCSV作成

7.1 open()

open(ファイル名, モード, encoding="utf-8")
モード 意味
"r" 読み取り
"w" 書き込み。既存ファイルは上書き
"a" 追記
"r+" 読み書き
"b" バイナリモード

7.2 with 文

with open("sample.txt", "r", encoding="utf-8") as f:
    data = f.read()
with を使うと、自動的にファイルが閉じられます。 f.close() を自分で書く必要がありません。

7.3 ファイルへの書き込み

with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello\n")
    f.write("Python\n")
"w" は上書きです。 既に同名ファイルがある場合、元の内容は消えます。

7.4 ファイルの読み込み

全体を読む

with open("sample.txt", "r", encoding="utf-8") as f:
    data = f.read()

1行ずつ読む

with open("sample.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line)

7.5 strip() と split()

line = "apple,100\n"

line = line.strip()
data = line.split(",")

print(data)

出力：

['apple', '100']

数値として使いたい場合は型変換します。

name, price = line.split(",")
price = int(price)

7.6 簡単なCSV作成

data = [
    ["name", "score"],
    ["Alice", 80],
    ["Bob", 90]
]

with open("scores.csv", "w", encoding="utf-8") as f:
    for row in data:
        line = f"{row[0]},{row[1]}\n"
        f.write(line)

7.7 CSVの読み込み

with open("scores.csv", "r", encoding="utf-8") as f:
    for line in f:
        line = line.strip()
        row = line.split(",")
        print(row)

7.8 csv モジュール

単純なCSVなら split(",") でも処理できます。 しかし、値の中にカンマが含まれる場合は csv モジュールを使うのが安全です。 

import csv

with open("scores.csv", "r", encoding="utf-8", newline="") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)

CSVの書き込み

import csv

data = [
    ["name", "score"],
    ["Alice", 80],
    ["Bob", 90]
]

with open("scores.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.writer(f)
    writer.writerows(data)
CSVファイルを扱うときは、newline="" を指定するのが一般的です。

9. 最終チェックリスト

数値

  • □ / は常に float
  • □ // はフロア除算
  • □ % は剰余
  • □ ** はべき乗
  • □ += は自己代入の省略形

文字列

  • □ str はイミュータブル
  • □ \n, \t の意味
  • □ raw文字列 r"..."
  • □ スライスは終了位置を含まない
  • □ スライス範囲外はエラーにならない

リスト

  • □ list はミュータブル
  • □ append と extend の違い
  • □ remove は値を削除
  • □ pop は位置を削除して値を返す
  • □ sort() は戻り値 None

制御構造

  • □ while True は無限ループ
  • □ range は stop を含まない
  • □ break は内側のループを抜ける
  • □ continue は次の繰り返しへ
  • □ ループの else は break されなかった時だけ

コレクション

  • □ dict はキーと値
  • □ 辞書ループ中にサイズ変更しない
  • □ set は重複なし・順序なし
  • □ tuple はイミュータブル
  • □ 1要素タプルは (x,)

関数

  • □ return がないと None
  • □ デフォルト引数は定義時に評価
  • □ 可変デフォルト引数に注意
  • □ *args はタプル
  • □ **kwargs は辞書

ファイル・CSV

  • □ open(filename, mode, encoding)
  • □ "w" は上書き書き込み
  • □ with open(...) を使う
  • □ strip() は改行などを除去
  • □ 複雑なCSVは csv モジュールを使う
